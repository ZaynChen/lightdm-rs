# lightdm-rs
The Rust bindings of liblightdm-gobject

## Usage

cargo add:
```sh
cargo add lightdm
```

Cargo.toml:
```toml
[dependencies]
lightdm = "0.1.0"
# git version
# lightdm = { git = "https://github.com/ZaynChen/lightdm-rs.git", version = "0.1.0" }
```
